# haskellen
This repo provides a ready-to-use Haskell development environment using:

- VS Code Dev Containers
- GHCup (GHC, Cabal, HLS)

## 🚀 Getting started

1. Open in VS Code
2. Install "Dev Containers" extension
3. Click: "Reopen in Container"
